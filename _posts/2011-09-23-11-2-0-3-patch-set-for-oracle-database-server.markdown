Just a quick post that the 11.2.0.3 patch set for Oracle Database Server has been released for x86 and x86-64 platforms (other ports will soon follow). The patchset number is [10404530](https://updates.oracle.com/Orion/PatchDetails/process_form?patch_num=10404530) and is available for download from [My Oracle Support](https://support.oracle.com/).

Also be sure to check out the [11.2.0.3 New Features Guide](http://download.oracle.com/docs/cd/E11882_01/server.112/e22487/chapter1_11203.htm).
